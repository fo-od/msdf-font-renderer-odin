# msdf-font-odin

a.k.a msdfont

parse/render fonts from [msdf-atlas-gen](https://github.com/Chlumsky/msdf-atlas-gen)! in odin!
it also supports parsing .arfont files! (not feature complete though, can only handle one atlas texture)

recommended usage is to generate an atlas and json file using [msdf-atlas-gen](https://github.com/Chlumsky/msdf-atlas-gen)!

raylib renderer is included, with an example!
